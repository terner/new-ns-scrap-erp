1. กู้กรรมการ
กรรมการโอนเงินเข้าบัญชีบริษัท เพื่อเพิ่มเงินสดหรือเงินฝากให้บริษัทใช้จ่าย
Field ที่ต้องกรอก
  • กรรมการผู้จ่าย
  • บัญชีบริษัทที่รับเงินเข้า
  • จำนวนเงิน
  • วันที่รับเงิน
  • หมายเหตุ
ผลลัพธ์ในระบบ
  • เพิ่มเงินเข้าบัญชีบริษัท
  • สร้างรายการ Bank Statement ฝั่งเงินเข้า
  • สถานะ: รับเข้าแล้ว / ยกเลิก
2. เงินกู้ยืมค่าสินค้า
บริษัทโอนเงินให้กรรมการ เพื่อให้กรรมการนำไปใช้ซื้อสินค้าแทนบริษัท
Field ที่ต้องกรอก
  • บัญชีบริษัทที่โอนออก
  • บัญชีธนาคารกรรมการที่รับเงิน
  • วงเงินกรรมการคงเหลือ
  • จำนวนเงิน
  • วิธีคืนเงิน: คืนตรง / ไม่คืน
  • เหตุผล กรณีไม่คืน
ผลลัพธ์ในระบบ
  • ส่งข้อมูลไปหน้าจ่ายเงิน
  • หักเงินจากบัญชีบริษัทเมื่อจ่ายจริง
  • ติดตามยอดค้างคืนของกรรมการ
3. สำรองจ่ายพนักงาน
บริษัทจ่ายเงินให้พนักงานล่วงหน้า เพื่อนำไปใช้จ่ายตามงานที่ได้รับมอบหมาย
Field ที่ต้องกรอก
  • พนักงานผู้รับเงิน
  • บัญชีพนักงาน
  • บัญชีบริษัทที่จ่ายออก
  • จำนวนเงิน
  • วัตถุประสงค์ / หมายเหตุ
ผลลัพธ์ในระบบ
  • ส่งข้อมูลไปหน้าจ่ายเงิน
  • ติดตามยอดรอคืนหรือรอเคลียร์ค่าใช้จ่าย
  • สถานะ: รอคืน / คืนแล้ว / ยกเลิก
ข้อเสนอ UX: Label ของช่องค้นหาผู้เกี่ยวข้องต้องเปลี่ยนตามประเภท เช่น กู้กรรมการ = “กรรมการผู้จ่าย”, เงินกู้ยืมค่าสินค้า = “กรรมการผู้รับเงิน”, สำรองจ่าย = “พนักงานผู้รับเงิน”
