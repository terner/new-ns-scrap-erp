# บันทึกความคืบหน้าการทำงาน (18 มิถุนายน 2026)

การทำงานในวันนี้มุ่งเน้นการแก้ปัญหาระบบใบชั่งน้ำหนัก (Weight Ticket) และการแยกส่วนข้อมูล Master Data ของ **"สินค้าสิ่งเจือปน"** ออกจากหน้าจอสินค้าหลักอย่างสมบูรณ์ เพื่อลดความสับสนในการใช้งานและอำนวยความสะดวกในการรับซื้อวัสดุสิ่งเจือปนตามความต้องการของธุรกิจ

---

## งานที่ดำเนินการสำเร็จแล้วในวันนี้

### 1. ระบบจัดการสิ่งเจือปนย่อยในใบชั่งน้ำหนักและการรับซื้ออัตโนมัติ (Weight Ticket Impurity Auto-Buy)
* **ปรับปรุง Layout แถวสิ่งเจือปนย่อย**: ปรับการจัดวางกล่องอินพุตในส่วนรายละเอียดขวาให้แสดงผลในระนาบขอบบนเสมอกัน (`items-start`) บน Desktop โดยนำ Label ในแถวย่อยออกและย้ายมาแสดงที่ส่วนหัวของกลุ่มรายการ (Table Column Headers) ส่งผลให้เวลาเกิดข้อความแจ้งเตือน (Error text) สีแดงยื่นลงด้านล่าง ตัวช่องกรอกและปุ่มจะไม่โยกขยับขึ้นข้างบนเลย
* **ระบบรับซื้อสิ่งเจือปนอัตโนมัติ (Auto-Purchase Match)**:
  - เพิ่มปุ่ม "ซื้อ" บนแถวสิ่งเจือปนย่อย เพื่อแปลงน้ำหนักหักสิ่งเจือปนนั้นไปเป็นล็อตรับซื้อใหม่ในบิล
  - เมื่อคลิกปุ่มระบบจะทำรายการจับคู่อัตโนมัติ (เช่น ขยะ จับคู่กับสินค้า บอร์ดขยะ) หากแมตช์พบสินค้าที่ตรงกันระบบจะสร้างล็อตใหม่เข้ามาในใบชั่งน้ำหนักทันทีโดยไม่เด้ง Dialog ยืนยันเพื่อความสะดวกรวดเร็ว
  - หากเป็นกรณีจับคู่ไม่พบ หรือเลือกเป็นประเภท "อื่นๆ" ระบบจะแสดง Dialog คอนเฟิร์มการรับซื้อพร้อมระบุฟิลด์บังคับค้นหาเฉพาะสินค้าประเภท "สินค้าสิ่งเจือปน" เท่านั้น
* **คัดแยกกลุ่มสินค้าในหน้าใบชั่งน้ำหนัก**:
  - แบ่งลิสต์สินค้าในหน้า WTI ออกเป็น `normalProducts` (สินค้าปกติ) สำหรับช่องค้นหาหลักเพื่อไม่ให้สินค้าสิ่งเจือปนปะปน
  - และ `impurityProducts` (สินค้าสิ่งเจือปน) สำหรับลอจิกจับคู่ซื้อและ Dialog คอนเฟิร์มการซื้อสิ่งเจือปน

### 2. การสร้างหน้าจอ Master Data สำหรับสินค้าสิ่งเจือปนโดยเฉพาะ
* **สร้าง Route หน้าเพจใหม่**:
  - เพิ่ม Route `/master-data/impurity-products` (หน้าเข้าที่ [page.tsx](file:///c:/new-ns-scrap-erp/apps/next/src/app/master-data/impurity-products/page.tsx))
  - พัฒนา Client Component [ImpurityProductsPageClient.tsx](file:///c:/new-ns-scrap-erp/apps/next/src/components/master-data/impurity-products/ImpurityProductsPageClient.tsx) เพื่อบริหารจัดการสินค้ากลุ่มนี้โดยเฉพาะ
* **ควบคุมสิทธิ์และความปลอดภัย**:
  - ล็อกประเภทสินค้าในฟอร์มของหน้าใหม่ให้เป็น `"สินค้าสิ่งเจือปน"` และเรนเดอร์ในแบบ Read-only เสมอ ป้องกันผู้ใช้สลับไปเลือกประเภทอื่น
  - เพิ่มการตั้งค่าระดับ Permission ใน [navigation.ts](file:///c:/new-ns-scrap-erp/apps/next/src/lib/navigation.ts) ให้หน้า `/master-data/impurity-products` ตรวจสอบสิทธิ์ผ่านสิทธิ์ `'master.products.view'` เช่นเดียวกับหน้าสินค้าปกติ
* **การซ่อนสินค้าสิ่งเจือปนในหน้าสินค้าปกติ**:
  - ปรับปรุง [ProductsPageClient.tsx](file:///c:/new-ns-scrap-erp/apps/next/src/components/master-data/products/ProductsPageClient.tsx) เพื่อกรองและซ่อนสินค้ากลุ่ม "สินค้าสิ่งเจือปน" ออกจากการแสดงผลทั้งหมด ทำให้ข้อมูลแยกออกจากกันอย่างเด็ดขาดในฝั่งผู้ใช้งาน
* **เพิ่มป้ายเมนูในแถบ Sidebar**:
  - เพิ่มลิงก์เมนูย่อย "รายการสินค้าสิ่งเจือปน" ใต้หมวดหมู่ "สินค้า" ของเซกชันข้อมูลหลักใน Sidebar เพื่อให้ผู้ใช้สามารถคลิกเข้าสู่หน้าจัดการข้อมูลได้อย่างสะดวก
